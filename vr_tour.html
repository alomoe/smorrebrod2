Добро пожаловать в виртуальный тур гастробара "Сморреброд"

Наш тур поможет вам познакомиться с интерьером, едой и напитками заведения.

Для осмотра локации поворачивайте телефон вокруг своей оси или вращайте локацию вручную с помощью пальцев.

Для перемещения по локациям необходимо найти места для перемещения и нажать на них.

Для ознакомления с меню необходимо познакомиться с барманами у бара на третьей локации. Для перехода на локацию ищи подсказку "Посмотреть меню".
